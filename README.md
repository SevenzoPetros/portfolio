# Sevenzo Petros Kudakwashe - Portfolio Website

A modern, responsive portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Modern Design**: Clean, colorful interface with blue-cyan-purple gradient theme
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Smooth Interactions**: Hover effects, transitions, and micro-animations
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Dark Mode Support**: Automatic theme detection
- **Fast Performance**: Optimized build with Next.js 16

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Inter & Poppins (Google Fonts)
- **Icons**: Next.js Image optimization

## 📦 Installation & Setup

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🚀 Deployment

### Vercel (Recommended)
1. Push your code to GitHub
2. Connect your repository to [Vercel](https://vercel.com)
3. Deploy automatically

### Other Platforms
```bash
# Build for production
npm run build

# Start production server
npm start
```

## 📝 Customization

### Update Personal Information
Edit `app/page.tsx` to update:
- Name and title
- Contact information (email, phone)
- About section content
- Projects showcase

### Add Your Photo
1. Place your photo in `public/` folder
2. Update the image source in the About section

### Customize Colors
Edit `app/globals.css` to modify the color scheme and gradients.
